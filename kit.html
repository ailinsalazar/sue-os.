Kitsord

  
La aplicación fue creada por los guatemaltecos Charlie Galdámez, Kevin Sánchez,
 Luis Pedro Méndez, Carlos Villagrán y Diego Andrés Carranza 
    
  Esta aplicación tiene como intención principal enseñar a las personas el lenguaje de señas guatemalteco. 
Actualmente la aplicación está únicamente para el sistema operativo
 Android y puedes descargarla gratuitamente.
